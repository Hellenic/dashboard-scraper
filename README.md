# dashboard-scraper [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Data scraper for the Dashboard API

## Usage

Build changes with:
```
gulp
```
Run tests with:
```
gulp test
```

Install dependencies with:
```
npm install
```

Run scraping with
```
npm run scrape
```
